# **Lab 3 - List ADT (Array Implementation)**

## **Overview**  
This program implements the List ADT using an array of size 5. The program provides various insertion, deletion, and utility operations using a menu-driven approach while maintaining boundary conditions.

## **Program - List ADT using Array**

### **Overview**  
This program manages a list using a fixed-size array and supports common List ADT operations such as insert, delete, search, display, and rotate.

### **Supported Operations**  
This is a menu-driven program that supports the following operations:

- **Insert Beginning** — Insert an element at the start of the list  
- **Insert End** — Insert an element at the end of the list  
- **Insert Position** — Insert an element at a specific position  
- **Delete Beginning** — Remove the first element of the list  
- **Delete End** — Remove the last element of the list  
- **Delete Position** — Remove an element at a specific position  
- **Search** — Search for an element in the list  
- **Display** — Display all elements of the list  
- **Rotate** — Rotate the elements of the list to the left or right  
- **Exit** — Terminate the program
