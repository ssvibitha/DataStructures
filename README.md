# Data Structures Lab

This repository contains complete set of programs implemented for the Data Structures Lab covering topics such as arrays, linked lists, stacks, queues, hashing, trees, and graphs.

## Tools and environment

Language: C++

Compiler: GCC compiler

## To compile the program use the command, 

g++ -o fileObject.exe file.cpp

## To run the compiled program use the command, 

./fileObject.exe
